# soon-ssh
귀찮음을 조금은 덜어줄 간단한 ssh 접속 도우미

# 적용 방법
1. `keys/` 에 접속명으로 ssh 키파일 저장 (.key)
2. `conf` 파일 작성 → `soon-ssh.sh` 실행 → 접속명 입력

# conf 예시
```bash
#$접속명1=$계정명1@$호스트2  
#$접속명2=$계정명1@$호스트2  
```
